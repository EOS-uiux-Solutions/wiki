# Welcome to the EOS project wiki

This Wiki defines the Design System used to deliver UX, UI, front-end, and other deliverables for the EOS project found at https://github.com/EOS-uiux-Solutions

## EOS product vision

*To deliver tools and solutions for front-end developers and UX designers to efficiently deliver and implement industry-standard practices that will help brands achieve consistency and success.*
